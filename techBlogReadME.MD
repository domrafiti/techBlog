# Project Title: Tech Blog

  [![License: Unlicense](https://img.shields.io/badge/license-Unlicense-blue.svg)](http://unlicense.org/)

## Description 

this is a CMS style blog that allows a user to create a profile, create and manage postings, and leave comments for others. It also is designed to log a user out if they are idle for 15 seconds.
